Forage Notes

Forage Notes is a personal project which is a collaboration with a digital designer. Working on this project has allowed me to research and begin to learn different technologies and languages that I may not normally use, such as Craft Cms, Twig, composer (to install Craft) and this project also helped me learn how to set up a gulpfile from scratch too.

While we finalise parts of the design you can view the holding page to find out more about Forage Notes.
